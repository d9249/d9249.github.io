## 목적
- 시간과 장소에 구애받지 말고 집에서 운동하자!
- AI를 이용한자세 추정을 통해 모든 기기로 해당 웹 서비스에 접속하여 운동

## 기능
- 자세 추정을 통한 운동 자세 교정
- 업적, 월간/주간 분석, 운동 기록 확인, 운동 영상 공유
- 지역 트레이너와의 채팅을 통한 온/오프라인 Meet-Up

## 개발환경
- FrontEnd : HTML, CSS, React
- BackEnd : Node.js MongoDB
