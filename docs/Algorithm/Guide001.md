# ICPC

CPC 처음 보시는분들 위해서 대강 어떻게 진행하는지 맛만 보시라고 연습 방법 알려드림서울와서 당황하지 마시고 미리 연습해보셈. 혼자 푸는 것과 3명이 푸는 건 느낌이 다름

# 준비할 것

프린트 할 수 있으면 노트북 한 대와 프린터, 문제와 팀노트를 프린트한 종이, 볼펜, 이면지프린터가 없으면 노트북 3대 가져와서, **문제를 풀 한 대를 정해놓고** 나머지는 문제와 팀노트만 켜놓기

인터넷 예선 준비면 3시간, 본선 준비면 5시간동안 풀면 됩니다.

# 풀고 나서

중간에 문제를 못 풀고 쉬는 시간이 있었는지, 못 푼 문제를 너무 오래 문제를 붙잡고 있지 않았는지 등을 생각해봅시다.어떻게 하면 3명이 효율적으로 문제를 풀 수 있을지 고민해봅시다.팀노트에 넣어야 할 내용이나 팀에서 부족한 부분이 있는지 생각해봅시다.공식 사이트에서 풀이를 찾아보고, 해당 대회의 스코어보드와 성적을 비교해봅시다.
